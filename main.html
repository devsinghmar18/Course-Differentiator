    DBMS (Database Management System) is software used to organize, store, and retrieve large amounts of data efficiently. It provides a mechanism for data retrieval, storage, and manipulation using SQL (Structured Query Language) or other programming interfaces.

    blockchain

    Blockchain is a decentralized, secure and transparent ledger technology. It consists of a chain of blocks that store data in a secure and tamper-proof way. Each block in the chain contains a unique cryptographic hash, which links it to the previous block and ensures that the data cannot be altered or deleted.
